# GHL Terminology

Use HighLevel terms first. Use Salesforce terms only as a private comparison after the GHL concept is clear.

## Core Terms

| HighLevel Term | Meaning | Avoid Saying First |
|---|---|---|
| Agency account | Top-level account, often used by an agency or main owner | Org |
| Sub-account / Location | A business/client workspace inside HighLevel | Org / business unit |
| Contact | Central person record in HighLevel | Lead record |
| Tag | Label used to group, identify, segment, or trigger automation for Contacts | Campaign/status/picklist |
| Custom Field | Extra information stored on a Contact or other supported record | Field |
| Pipeline | Visual process for tracking Opportunities | Sales Process |
| Pipeline Stage | Step inside a Pipeline | Stage |
| Opportunity | Lead/deal card inside a Pipeline | Opportunity record |
| Workflow | Automation builder in HighLevel | Flow |
| Workflow Trigger | Event that starts a Workflow | Flow trigger |
| Workflow Action | Step performed by a Workflow | Flow action |
| Form | Lead capture form | Web-to-Lead |
| Survey | Multi-step lead capture or qualification tool | Web-to-Lead form |
| Funnel | Landing page or multi-step conversion path | Landing page / Experience page |
| Calendar | Booking setup for appointments | Scheduler |
| Appointment | Booked meeting or booking record | Event |
| Conversations | Communication and message history area | Activity Timeline |
| Snapshot | Reusable configuration package/template | Change Set / metadata package |
| Integration | Connected external app or service | Connected App |
| Webhook | Outbound or inbound technical event/API-style call | API callout |

## Practice Sentences

Use these sentences until they feel natural:

1. A Contact is the central person record in HighLevel.
2. A Tag can identify, group, or trigger automation for a Contact.
3. A Pipeline tracks Opportunities through Pipeline stages.
4. A Workflow can apply a Tag, send a message, create an Opportunity, or move an Opportunity.
5. A Form can capture a lead and create or update a Contact.
6. A Calendar is used to book Appointments.
7. Conversations show communication history for a Contact.
8. A Snapshot can copy selected setup/configuration into another sub-account/location.

## One-Sentence Lead Journey

A Contact enters HighLevel through a Form, a Workflow applies a Tag and creates an Opportunity in a Pipeline, and activity can be reviewed through Conversations and Contact activity.
